# Projeto Coruja — Build 004.1 Fundação

## Como atualizar
1. Faça backup da pasta atual.
2. Substitua os arquivos pelos arquivos desta pasta.
3. Publique novamente no GitHub Pages.
4. Abra o sistema e faça um teste com poucos jogadores antes da rodada oficial.

## Novidades
- Registro de gols.
- Histórico permanente de rodadas.
- Estatísticas básicas.
- Encerramento seguro: primeiro salva e valida, depois limpa.
- Financeiro com mensalistas, avulsos e isentos.
- Proteção contra encerramento sem sorteio.
- Correção preservada: jogador que chega depois do sorteio recebe atraso.

# Projeto Coruja — Build 002 Dashboard

## Versão
1.1.2-build002-dashboard

## Alterações
- Dashboard inteligente na aba Rodada.
- Indicadores de presentes, confirmados, times previstos, equilíbrio, receita e atrasos.
- Resumo automático da rodada.
- Barra de progresso dos presentes.
- Status visual do Firestore.
- Engine de sorteio preservada.
- Validações e backup da v1.1.1 preservados.

## Instalação
Substitua somente `index.html`, `style.css` e `app.js` dentro da pasta `public`.
Antes, mantenha uma cópia de segurança dos arquivos atuais.
Depois, atualize o navegador com Ctrl+F5.

# Projeto Coruja — Build 002 Financeiro

## Versão
1.1.3-build002-financeiro

## Novidades
- Caixa do Sub Óbito.
- Locação fixa mensal de R$ 1.000,00.
- Registro de goleiro por aplicativo.
- Registro de rateio de bola.
- Outras despesas variáveis.
- Resumo de receitas, despesas e saldo.
- Prestação de contas pronta para WhatsApp.
- Dados financeiros salvos localmente neste aparelho.
- Dashboard e engine de sorteio preservados.

## Instalação
Substitua `index.html`, `style.css` e `app.js` dentro da pasta `public`.
Depois execute o deploy e atualize o navegador com Ctrl+F5.
